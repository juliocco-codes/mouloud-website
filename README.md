# Travels Holidays in Morocco

Static website for **Travels Holidays in Morocco**, a tour guide business run by Mouloud Tazint, a certified mountain guide based in Marrakech.

## What this is

A single-page static website showcasing Mouloud's tour offerings across the Sahara Desert, Atlas Mountains, and Atlantic Coast. No framework, no build step — just HTML, CSS, and a few lines of JavaScript.
